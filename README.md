# Quickstart


## 1. Activate `env` 
> **Windows**

 ```cmd
 env\Scripts\activate.bat
 ```

> **Mac/Linux**
```Terminal
source env/bin/activate
```


## 2. Install requirements
`pip install -r requirements.txt`


## 3. Migrate
`python manage.py migrate`


## 4. Run server
`python manage.py runserver`
